out2asm:
Original author of the C code : ?
Delphi sources by myself (Chloé Avrillon)

usage :
out2asm filein.ext fileout.ext

if extension is ommitted, default extension .out for input file and .s for output file are used.
only one filename, with or without extension can be provided.

out2asm filename
will convert filename.out to filename.s
